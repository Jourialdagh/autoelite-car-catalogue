How to Browse the Catalogue

Using the Search Bar

Type any brand name or model into the search bar at the top of the catalogue page. Results update in real time.

Filter by Brand

Use the Brand dropdown to show only vehicles from a specific manufacturer like BMW, Toyota, or Tesla.

Filter by Type

Select a vehicle type — Sedan, SUV, Electric, Sports, Truck, etc. — to narrow results.

Price Range Slider

Drag the price slider to set a maximum budget. All vehicles above the limit are hidden automatically.

Resetting Filters

Click the Reset button to clear all filters and show the full catalogue again.

Viewing Product Details

Click 'View Details' on any card to see full specs, options, and features for that vehicle.
